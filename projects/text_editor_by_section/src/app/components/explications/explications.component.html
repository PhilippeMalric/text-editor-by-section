Bienvenue sur l'application web du rendez-vous 2021 de l'ACCQ

Authentification / Inscription

Ordinateur

Pour vous inscrire, allez dans la barre en haut, vers la droite et cliquez sur : «Connexion».
Si c'est la première fois, vous devrez créer un compte et le vérifier en cliquant sur le lien envoyé. (Création d'un compte)
Puis vous connecter.
Les fois suivantes, vous pourrez vous connecter avec votre mot de passe. (Authentification)

Téléphone ou une tablette

Sur un téléphone ou une tablette, cliquez sur les 3 barres horizontales, en haut à gauche.

Oublie de votre mot de passe

Si vous oubliez votre mot de passe, vous devrez inscrire votre courriel et cliquez sur : «Réinitialiser le mot de passe» dans le bas de la page de connexion.
Ensuite, allez le réinitialiser à partir du lien envoyé à votre courriel.

La page Quiz

L'onglet : Quiz sera débaré lorsque vous serez authentifié.
